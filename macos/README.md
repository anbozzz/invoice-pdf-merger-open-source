# 发票 PDF 合并工具

一个最小版 macOS 本地工具，用于导入多张 PDF 发票，并按列表顺序合并成一个 PDF 文件。

## 功能

- 多选导入 PDF
- 支持拖拽 PDF 到窗口
- 支持从 QQ、网易 163、网易 126、网易 yeah.net 邮箱下载近 N 日 PDF/ZIP 附件
- 支持 12306 ZIP 附件自动解压并提取 PDF
- 支持剔除滴滴等行程单、行程明细类 PDF
- 邮箱地址和授权码会按邮箱类型保存到 macOS Keychain，下次自动填充
- 邮箱导入成功后的 PDF 自动加入合并列表
- 支持在列表中直接预览 PDF，不调用外部应用
- 支持选中文件后按空格键预览
- 显示文件名、页数、文件大小
- 支持上移、下移、删除、清空
- 合并后保存到本地
- 全程本机处理，不上传文件

## 邮箱导入规则

首次使用前，需要在邮箱网页端开启 IMAP 服务，并生成授权码或客户端授权码。应用内填写的是“授权码”，不是邮箱登录密码。

默认处理规则：

- 只扫描近 3 日邮件，可在导入弹窗中调整为 1-30 日。
- 下载 `.pdf` 和 `.zip` 附件。
- `.zip` 附件会自动解压，适用于 12306 发票压缩包。
- 解压后的 PDF 会提取到 `Downloads/发票PDF合并/邮箱导入-日期时间/`。
- 文件名或 PDF 内容包含“行程单、行程明细、itinerary、trip”等特征时，默认剔除。
- 筛选后的 PDF 会自动加入合并列表。
- 邮箱地址和授权码按邮箱类型保存到 macOS Keychain。
- 可在邮箱导入弹窗中清除当前邮箱类型保存的账号信息。

## 运行

在本目录执行：

```bash
swift run
```

## 打包为可安装 DMG

在本目录执行：

```bash
bash scripts/build_dmg.sh
```

生成结果：

- `dist/发票PDF合并.dmg`

打开 `.dmg` 后，把应用拖到 `Applications` 即可安装。

## 系统要求

- macOS 13 或更高版本
- Xcode Command Line Tools
