# Security

## Sensitive Data

Do not commit real mailbox addresses, QQ Mail authorization codes, invoices, or downloaded attachments.

The application is designed to process files locally. QQ Mail credentials are stored on the user's own computer:

- macOS: Keychain
- Windows: Electron safe storage

## Reporting Issues

If you find a security issue, please do not publish real invoice files or credentials in a public issue. Open a minimal report with reproduction steps and redact all personal data.
